# Etch-a-Sketch

A project built as part of [The Odin Project](https://www.theodinproject.com/) (Foundations).

## Description
An interactive grid inspired by the Etch-A-Sketch toy: the mouse "draws" by changing the color of each square on hover.

## Technologies
- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation)
